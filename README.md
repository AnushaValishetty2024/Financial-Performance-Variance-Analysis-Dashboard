# Financial-Performance-Variance-Analysis-Dashboard
Developed a Power BI dashboard to analyze actual vs. budget performance using KPI cards, DAX measures, and interactive visualizations. Enabled variance analysis, trend monitoring, and category-wise performance tracking for business decision-making.

# Project Objective
To analyze actual sales performance against budgeted targets and identify financial performance gaps, trends, and variance drivers through interactive Power BI reporting.
