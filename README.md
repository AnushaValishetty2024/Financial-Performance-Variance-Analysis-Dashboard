# Financial-Performance-Variance-Analysis-Dashboard
Developed a Power BI dashboard to analyze actual vs. budget performance using KPI cards, DAX measures, and interactive visualizations. Enabled variance analysis, trend monitoring, and category-wise performance tracking for business decision-making.

# Project Objective
To analyze actual sales performance against budgeted targets and identify financial performance gaps, trends, and variance drivers through interactive Power BI reporting.

# Questions (KPIs)
- What is the Total Sales achieved?
- What is the Total Budget allocated?
- What is the Sales Variance?
- What is the Variance Percentage?
- Which category performs above or below budget?
- How do sales trends change over time?

# Process
- Imported Actual, Budget, and Date datasets into Power BI.
- Performed data cleaning and transformation using Power Query.
- Built relationships between tables using data modeling.
- Created DAX measures for Sales, Budget, Variance, and Variance %.
- Developed interactive dashboards with KPI cards, charts, and slicers.
